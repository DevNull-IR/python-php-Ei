# چاپ محتوی :

**Python in python-php-Ei**

```python 
echo ('محتوی تست')
```


**php**

```php
echo ('محتوی تست');
```

یا
```php
echo 'محتوی تست';
```

# تابه substr در php :

```php 
echo substr('This Text Test in Substr function',0,5);
```


و در فانکشن های ما در پایتون :

```python 
echo(substr('This Text Test in Substr function',0,5))
```


# تابع Print_r در php :

```php

print_r(['one'=>'tow']);
```



در پایتون :
```python 
print_r({1,2,3})
```



# ایجاد یک فایل جدید

در php :



```php 

file_put_contents('file.nm',null,8);
```


در پایتون : 
```python 

file_put_contents('file.nm',None,8)
```
 
# فانکشن strlen :

در php :


```php
echo strlen("Test");
```
در پایتون : 

```python
echo( strlen("Test"))
```

# ایجاد فایل جدید

در php :
```php
file_put_contents('Test.txt','متن من');
```
اضافه کردن به آخر فایل :
```php
file_put_contents('Test.txt','M TEXT',8);
```

**در پایتون**

```php
file_put_contents('Test.txt','متن من')
```
اضافه کردن به آخر فایل :
```php
file_put_contents('Test.txt','M TEXT',8)
```

# حذف فایل : 
در php
```php
unlink('file name');
```
در پایتون :

```python 
unlink('file name')
```
